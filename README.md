Face Recognition Attendance System 🎯

This project is a Real-time Face Recognition Attendance System built using Python, OpenCV, and face_recognition library that detects faces through a webcam and automatically marks attendance in a CSV file.

This project was developed as Internship Task-2 at Skilllnfytech.

---

🚀 Features

- Real-time face detection using webcam
- Face recognition with confidence score
- Automatic attendance marking
- CSV attendance storage
- Screenshot capture support
- Simple and user-friendly interface

---

🛠️ Technologies Used

- Python
- OpenCV
- face_recognition library
- NumPy

---

📂 Project Structure

FaceRecognitionAttendance/

├── attendance_system.py

---

▶️ How to Run the Project

Step 1:

Install dependencies

pip install -r requirements.txt

Step 2:

Register faces

python register_faces.py

Step 3:

Run attendance system

python attendance_system.py

---

📊 Output

The system automatically creates a CSV file and stores attendance records with name and timestamp.
